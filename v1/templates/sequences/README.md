# Sequence Diagram

Dokumen ini hanya membantu merender template sequence ke bentuk diagram.

Baca aturan dan registry sequence utama pada [docs/architecture/sequence-pattern.md](../../docs/architecture/sequence-pattern.md).

&nbsp;

## Usage

1. Copy sequence text dari template yang ingin digunakan.
2. Buka https://sequencediagram.org/.
3. Paste sequence text.
4. Aktifkan Presentation Mode bila diperlukan.
